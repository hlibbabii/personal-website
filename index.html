About Me

I am studying language models for source code and their applications to different software engineering tasks such as code completion, identification of buggy code and others. Most recently i was looking into the problem of modeling source code vocabulary and analyzing its impact on the performance of neural language models trained on code corpora pre-processed according to different modeling choices.

Also, I am a big fan of code quality and following best practices for writing clean code.
